# afrocomposer-hub
The ultimate platform for independent artists. Distribute, master, and monetize your music with 100% free distribution, unlimited downloads, and real-time fan links.

## Features
- **Pro Distribution:** Seamlessly upload WAV/MP3 files and manage ISRC/UPC codes.
- **PushPromo Hub:** Create unlimited smart links and pre-save campaigns.
- **Analytics:** Track fan engagement in real-time.
- **SEO Ready:** Auto-generated metadata for maximum discoverability.

## Built With
- [Tailwind CSS](https://tailwindcss.com/)
- [FontAwesome](https://fontawesome.com/)
